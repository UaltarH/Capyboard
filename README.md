# Capyboard
___

A shared dashboard for personal use that helps you keep everything organized

## Features
- Connected Calendar with private and shared events
- Todo, Wishlist
- Notes, Snippets
- Bookmarks
- Shop list and recipe list
- Budget management
- And more ..

## Front :
- VueJS
- ChartJs
- FullCalendar.io
- PrimeVue
- GridStack Vue

## Back :

- PostgreSQL
